# C语言学习笔记

## 第一章 程序设计和C语言

1.计算机程序是计算机能够识别和执行的指令

2.计算机语言

​	(1)机器语言：就是0和1组成的计算机可以识别的语言

​		机器指令：计算机能够直接识别和接受的二进制代码		

​	(2)符号语言(汇编语言)：通过符号表示二进制数。

​		汇编语言通过<u>汇编程序</u>可以转换为机器语言。

​		汇编语言的缺点是：不同计算机之间不通用，非常依赖于硬件

​	(3)高级语言：使用很接近人类的语言完成计算机程序的设计

​		高级语言的特点：接近人类语言，不依赖于机器。

​		高级语言通过**编译程序**可以转化为机器语言

​		高级语言的发展经过了<u>非结构化语言</u>、<u>结构化语言</u>、<u>面向对象语言</u>三个阶段。

3.c语言的标准：目前有C89和C99两个标准，常用C99标准。

4.c语言的特点：c语言是面向过程的语言，可开发应用程序和系统软件。

5.第一个C语言程序

```
#include<stdio.h>				//导入标准输入输出函数库standard input output 
int main(){						// main函数 程序的入口 int表示main函数返回值的类型 
	printf("Hello World!\n");	//打印到控制台 
	return 0;					//正常结束返回0,异常结束返回非0的整数 
}
```

​	每一个C语言程序有且只有一个main函数（主函数）。

​	C语言的注释有两种方式：

​	（1）//为行注释（仅C99标准支持）

​	（2）/*  */为段注释。

6.C语言的程序结构

​	（1）一个C语言程序是由一个或多个源程序文件组成的；

​	（2）一个从语言程序由一个或多个函数组成，其中有且只有一个main函数；

​	（3）C语言总是从main函数开始执行；

​	（4）每个C语言语句后面必须有一个（;）；

​	（5）C语言语句本身是不包含输入（scanf）输出（printf）函数的，是通过stdio.h引入进来的.
